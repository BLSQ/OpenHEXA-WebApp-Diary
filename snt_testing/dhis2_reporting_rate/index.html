Pipeline

DHIS2 Reporting Rate

Calculates facility reporting rates by data element based on routine data.

Parameters
Routine data source required
Select which routine data to use. 'raw' loads formatted routine data, 'imputed' loads outliers-imputed routine data, 'outliers_removed' loads routine data with outliers removed.
Facility Activity indicators required
Define which data elements will be used to determine the activity of a facility. A facility is considered 'active' if at least one of these indicators has a non-missing value greater than zero.
Volume activity indicators required
Define which data elements will be used to determine the volume of activity at a facility. Volume of activity is used to calculate WEIGHTED reporting rates.
Denominator method required
How to calculate the total number of facilities expected to report.
Outputs
