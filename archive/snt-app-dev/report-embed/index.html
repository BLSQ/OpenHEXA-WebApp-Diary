SNT — HTML Report Embed Probe

Diagnostic for PRODUCT_SPEC §9.1 "Report viewing" / open decision #10. Determines whether a pipeline run's HTML report (a bucket object) can be embedded in-app. It mints a signed prepareObjectDownload URL exactly as the orchestrator does, then runs two tests: (A) can it be shown in an <iframe src> (Option A), and (B) can it be fetched cross-origin for srcdoc/Blob injection (Option B). Needs scopes PIPELINES_READ + FILES_READ.

Scans recent runs of every pipeline and lists those whose outputs include an .html/.htm bucket object.
