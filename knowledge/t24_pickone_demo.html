T2.4 — "pick one" mutual-exclusion: visual treatment

Within an alternative group, the current member is the one with the most recent successful run (green radio ●, green ring, "active" tag). The others are superseded: dimmed but still fully clickable / runnable (hollow radio ○). A running member shows its live badge but does not supersede yet; a failed member changes nothing — the previous success stays current.

Variant A — group box keeps "choose one"; the radios carry the state

Quieter. The filled vs hollow radio + the dimming communicate the pick; the box label stays generic.
